**SmartMedicine-智能医疗App**
======================

# 项目介绍

智能医疗App:
* 包括:文章推荐,医疗社区,动态发布,在线问诊,AI问答,疾病预测,便捷搜索等功能
* 推荐算法：协同过滤，基于内容，神经网络
* 搜索引擎：ElasticSearch、word2vec/IK分词搜索、知识图谱
* IM系统：SpringCloud系列，Netty
* 终端：Android、JNI
* 大数据：Hadoop、Spark、Flink、Hive、HBase
* 爬虫：selenium、lxml
* 后台：JS，Vue，ElementUI
